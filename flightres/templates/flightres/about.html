UAV Guidelines

Drone Registry


Drone Registry is a platform to register for UAVs flights. According to the latest UAV guidelines UAV permission in Nepal has to be granted by multiple agencies and while there is no centralized repository for the collected data, this platform aims to act as a centralized database for the collected UAV data. This will aid the increase in the number of UAV registrations and flight permission retain this information.

A digital tool such as our platform will introduce flexibility and extensibility in the current system. The form created in our platform can be easily accessed through the messaging platform WhatsApp. Using WhatsApp has allowed us to provide users with a painless data entry experience. The information collected from the WhatsApp form is stored in a centralized database and despite the increasing numbers of new UAV registrations or flight permissions records it will still scale. Furthermore, a digital tool opens new opportunities to run analytics on the collected data.

Over the past several years there has been a surge in news events involving careless operators misusing drones, crashes between manned aircraft and UAVs. In an effort to educate new airspace users on the safe operation of UAS, digital tools come up to facilitate drone users' authority by centralizing data of each user in a single platform. There is no requirement to re-register UAS that have already been registered under this system. Operators may continue using the existing registration system or, alternatively, switch over to the new registration system when the registration is renewed.
